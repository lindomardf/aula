O que é Computação em Nuvem?

A computação em nuvem é um modelo de entrega de recursos de TI pela internet sob demanda. Ela permite o acesso remoto a softwares, armazenamento de dados e poder de processamento.

Modelos de Serviço

  • IaaS - Infraestrutura como Serviço (ex: AWS EC2, Azure VM)
  • PaaS - Plataforma como Serviço (ex: Heroku, Google App Engine)
  • SaaS - Software como Serviço (ex: Google Drive, Microsoft 365)

Benefícios da Nuvem

  1. Escalabilidade
  2. Redução de custos
  3. Acesso remoto
  4. Segurança e backup

Exemplos de Provedores de Nuvem

Amazon Web Services (AWS), Google Cloud Platform (GCP), Microsoft Azure, Oracle Cloud, IBM Cloud.
